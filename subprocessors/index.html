Sub-processors

Last updated: December 2025

This page lists third-party sub-processors used to operate Obscure Holiday Calendar services, including the website, mobile apps, and Slack/Discord bots.

Current sub-processors

Railway

Purpose: Hosting and infrastructure for bot services.

Slack

Purpose: Platform integration for the Slack bot, OAuth installs, and message delivery.

Discord

Purpose: Platform integration for the Discord bot and message delivery.

Stripe

Purpose: Payment processing for premium subscriptions.

Google (AdSense/AdMob/Analytics)

Purpose: Advertising, measurement, and analytics.

Updates

We may update this list as services change. Material changes will be reflected on this page.
